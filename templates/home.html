Welcome to BaySpec

A modern, browser-based fitting workbench for high-energy astrophysical spectra. Combine local, Astromodels, XSPEC and user-defined components, then sample with emcee, multinest, lmfit or iminuit — all from a single, reproducible interface.

What you can do

🔭

Bring your data

Upload OGIP-format src / bkg / rsp (or rmf + arf); pick statistic, noticing, grouping, rebinning per unit.

🌈

Compose your model

Select components from local / Astromodels / XSPEC, or paste your own Python class. Combine with + - * /.

📝

Run inference

Pair Data ↔ Model, manually fit, then run a Bayesian sampler or maximum-likelihood optimizer with one click.

Installation

BaySpec is on PyPI:
