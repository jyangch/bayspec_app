High-energy astrophysics

Welcome to BaySpec

A browser-based spectral fitting workbench. Load OGIP-like spectra, compose multi-component models from local, Astromodels, XSPEC or user-defined libraries, then run a Bayesian sampler or maximum-likelihood optimizer — all without leaving your browser.

Bring your data Upload OGIP-format src / bkg / rsp (or rmf + arf); pick statistic, notice energy, grouping, and rebinning per unit. Go to Data → › Compose your model Pick components from local / Astromodels / XSPEC, or paste your own Python class. Combine with + − × ÷. Go to Model → › Run inference Pair Data ↔ Model, manually fit, then run a Bayesian sampler or max-likelihood optimizer with one click. Go to Inference →
Installation

BaySpec is available on PyPI:

pip install bayspec

Optional backends

BackendInstallRole
emceepip install emceeMCMC (affine-invariant ensemble sampler)
