How to write a custom model

  1. Subclass Additive or Multiplicative from bayspec.model.model.
  2. Set self.expr (model name), self.comment (description), self.config (fixed settings as OrderedDict), and self.params (free parameters as OrderedDict of Par(value, prior)).
  3. Implement func(self, E, T=None, O=None) — return photon flux array evaluated at energy midpoints E (keV). T is time (seconds) for time-dependent models; O carries optional extra context.
  4. Click Register model. The class is available immediately in the Model page's user library dropdown.
