How to write a custom model
  1. Subclass Model from bayspec.model.model and call super().__init__().
  2. In __init__, set self.expr (the key used in compose expressions), self.type ('add' for additive, 'mul' for multiplicative, 'math' for pure-math), and self.comment. Populate self.config with Cfg(value) entries for fixed settings (e.g. redshift) and self.params with Par(value, prior) entries for free parameters — both as OrderedDict.
  3. Implement func(self, E, T=None, O=None). E is a 1-D array of energy bin midpoints in keV. Return a spectrum array of the same length: photon flux N(E) [photons/cm²/s/keV] for add type, or a dimensionless multiplier F(E) for mul/math type. T is a time array (seconds) only needed for time-resolved models; O is the nested model passed by a convolution operator.
  4. Click Register model. The class is available immediately in the Model page's user library dropdown.
